MCP-native browser for AI agents.

Read and act on any web page with your logged-in sessions — from Claude Code, Cursor, Codex, Windsurf, Trae, OpenClaw, and more.

Install Public source

What it does

  • Read any page — your agent reads WeChat articles, Zhihu threads, GitHub issues, Outlook inbox, the sites you're already logged into.
  • Act on any page — click, fill, navigate, submit. Your agent drives a real browser, not a screenshot-and-pray model.
  • Two editions — Lite installs from the Chrome Web Store. Full unlocks CDP-level control (focus emulation, user-gesture synthesis, network body capture) for long-running and gesture-gated tasks.

For whom

If you ship AI agents — IDE assistants, deep-research workflows, or full agent platforms — OMB is the browser execution layer that keeps your agent on the user's side of the login wall.

Install

One command, any shell:

curl -fsSL https://api.omb.org.cn/install/install.sh | sh

Or follow the guided install page.
