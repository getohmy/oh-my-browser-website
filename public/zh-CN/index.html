English

为 AI agent 打造的 MCP 原生浏览器。

用你已有的登录态读取和操作任意网页 —— 支持 Claude Code、Cursor、Codex、Windsurf、Trae、OpenClaw 等。

立即安装 公开源码

核心能力

  • 读任意页面 —— Agent 帮你读微信公众号、知乎回答、GitHub issue、Outlook 收件箱，所有你已登录的站点。
  • 操作任意页面 —— 点击、填表、导航、提交。Agent 驱动真实浏览器，而不是"截图+猜"。
  • 两个版本 —— Lite 从 Chrome 网上应用店一键装。Full 解锁 CDP 级控制（焦点模拟、用户手势合成、网络 body 抓取），用于长任务和需要手势的场景。

适合谁

如果你在做 AI agent 产品 —— 开发者工具、深度研究 workflow、或完整的 agent 平台 —— OMB 是让你的 agent 留在用户登录墙内侧的浏览器执行层。

一键安装

任何 shell，一条命令：

curl -fsSL https://api.omb.org.cn/install/install.sh | sh

或访问 安装指南页。
