隐私政策 — 摘要

权威文档：完整的法律文本在公开源码镜像的 store-assets/chrome-web-store/privacy-policy.md 。本页为执行摘要，以 markdown 文件为准。

我们**不**采集什么

  • 你访问的 URL
  • 页面内容、表单、或 agent 读取的任何内容
  • Cookie、session token、凭证
  • 浏览历史、书签、剪贴板
  • 可识别形式的 IP（仅存储用于限流审计的 hash 前缀）

我们采集什么

  • 账号数据：注册时的 email + OAuth 身份。用于提供服务，删除账户时一并删除。
  • 匿名遥测：任务类型 + 成败 + 时长 + 域名 hash + 错误码（固定枚举）。保留 90 天。无 URL，无页面内容。
  • 错误报告：扩展 / dashboard 的 JavaScript 异常，通过 beforeSend hook 剥离 PII，发到 Sentry。
  • 请求指标：聚合看板（成功率 + p95 延迟），仅按域名 hash 分组。
  • Site Memory：特定域名上结构级 CSS 选择器（已剔除任何识别内容）。完整披露见权威文档，包括 omb_memory(url) peek 端点 —— 它从不返回选择器字符串或页面内容。
  • 浏览器指纹：SHA-256(平台 + UA + 分辨率 + 色深 + 时区 + 语言 + CPU 线程数) 的 hash。用于你在多台机器装 OMB 时正确路由 agent 任务。GDPR 下作为假名个人数据处理。

我们永不做的事

  • 售卖或分享数据给第三方（仅有的第三方是基础设施：Cloudflare / Sentry）。
  • 跨服务用户画像。
  • 在你安装并登录前做指纹采集。

你的权利

从 dashboard 用户菜单删除账户。删除会级联到 devices、关联 user_id 的 telemetry 行、以及私有层 Site Memory 条目。
